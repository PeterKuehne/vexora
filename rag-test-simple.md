# KI und Machine Learning Guide

## Was ist RAG (Retrieval-Augmented Generation)?

RAG ist ein KI-Ansatz, der Informationsretrieval mit Textgenerierung kombiniert. Das System sucht zunächst relevante Dokumente aus einer Wissensbasis und verwendet diese Informationen dann, um kontextbezogene Antworten zu generieren.

## Wie funktioniert Weaviate?

Weaviate ist eine Open-Source-Vector-Datenbank, die für AI-Anwendungen optimiert ist. Sie ermöglicht es, unstrukturierte Daten als Vektoren zu speichern und semantische Suchen durchzuführen.

## Machine Learning Grundlagen

Machine Learning umfasst verschiedene Ansätze:

1. **Supervised Learning**: Lernen mit gelabelten Daten
2. **Unsupervised Learning**: Muster in ungelabelten Daten finden
3. **Reinforcement Learning**: Lernen durch Belohnung und Bestrafung

## Vector Embeddings

Vector Embeddings sind numerische Repräsentationen von Text, die semantische Bedeutungen erfassen. Sie ermöglichen es, ähnliche Konzepte im Vektorraum nahe beieinander zu platzieren.
