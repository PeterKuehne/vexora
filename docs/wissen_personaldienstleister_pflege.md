# Personaldienstleister im Gesundheitswesen

> Wissensgrundlage: Zeitarbeit in der Pflege, interne Strukturen, Rolle des Disponenten

---

## 1. Was ist das?

Ein **Personaldienstleister im Gesundheitswesen** ist ein Unternehmen, das Pflegefachkräfte bei sich anstellt und sie temporär an Kliniken, Pflegeheime und ambulante Dienste verleiht. Das rechtliche Modell heißt **Arbeitnehmerüberlassung (AÜG)** — umgangssprachlich "Zeitarbeit" oder "Leiharbeit".

### Das Dreiecksverhältnis

```
┌──────────────────────────────┐
│    Personaldienstleister     │ ← Arbeitgeber (zahlt Gehalt,
│         (Verleiher)          │   Sozialabgaben, Urlaub)
└──────┬───────────────┬───────┘
       │               │
  Arbeitsvertrag   Überlassungsvertrag
       │               │
       ▼               ▼
┌────────────┐   ┌─────────────────┐
│ Pflegekraft│   │ Klinik/Pflegeheim│ ← Einsatzbetrieb (Entleiher)
│ (Leiharbeit-│  │ (Kunde)          │   zahlt Verrechnungssatz
│  nehmer)   │   └─────────────────┘    an den Dienstleister
└────────────┘
  weisungsgebunden am Einsatzort
```

- Die **Pflegekraft** ist beim Personaldienstleister fest angestellt (unbefristeter Vertrag)
- Der **Kunde** (Klinik, Heim) zahlt einen Verrechnungssatz pro Stunde an den Dienstleister
- Die Pflegekraft arbeitet vor Ort beim Kunden, unterliegt dort dem **Weisungsrecht**

### Warum gibt es das?

Krankenhäuser und Pflegeeinrichtungen haben chronischen Personalmangel. Zeitarbeit dient als **Flexibilisierungsinstrument**:

- Ausfälle beim Stammpersonal kompensieren (Krankheit, Urlaub, Mutterschutz)
- Saisonale Spitzen abfangen (Grippewellen, Pandemien)
- Vakanzen überbrücken bis Festanstellung besetzt ist
- Neue Stationen/Bereiche hochfahren

**Größenordnung:** Von ~1,83 Mio. Pflegekräften in Deutschland sind nur ca. 23.800 (1,3%) bei Personaldienstleistern beschäftigt — aber ihr Einfluss auf die Versorgungssicherheit ist überproportional groß.

## 2. Rechtlicher Rahmen (AÜG)

| Regelung                    | Detail                                                                                |
| --------------------------- | ------------------------------------------------------------------------------------- |
| **Erlaubnispflicht**        | Personaldienstleister braucht eine AÜG-Erlaubnis der Bundesagentur für Arbeit         |
| **Überlassungshöchstdauer** | Max. 18 Monate beim selben Entleiher (danach Übernahme oder Pause ≥3 Monate)          |
| **Equal Pay**               | Nach 9 Monaten beim selben Kunden: gleiches Gehalt wie Stammpersonal                  |
| **Branchenzuschläge**       | Tarifvertragliche Stufenmodelle können Equal Pay auf 15 Monate strecken               |
| **Equal Treatment**         | Gleiche Arbeitsbedingungen (Kantine, Parkplatz, Arbeitskleidung)                      |
| **Kennzeichnungspflicht**   | Überlassung muss im Vertrag als solche gekennzeichnet sein                            |
| **Fachkraftquote**          | Einrichtungen müssen Mindest-Fachkraftquoten einhalten — Zeitarbeitskräfte zählen mit |

## 3. Das Geschäftsmodell — Kalkulation

### Verrechnungssatz

Der Kunde zahlt pro geleistete Stunde einen **Verrechnungssatz**. Dieser wird typischerweise als Multiplikator auf den Bruttolohn berechnet:

```
Verrechnungssatz = Bruttostundenlohn × Faktor (1,8 – 2,4)
```

### Kostenstruktur eines typischen Einsatzes

| Kostenblock                                          | Anteil        |
| ---------------------------------------------------- | ------------- |
| Bruttolohn der Pflegekraft                           | ~40–45%       |
| Sozialversicherung (AG-Anteil)                       | ~20%          |
| Rückstellungen (Urlaub, Krankheit, Feiertage)        | ~10–12%       |
| Verwaltung (Lohnbuchhaltung, Disposition, Miete, IT) | ~10–15%       |
| Recruiting (Stellenanzeigen, Bewerbungsprozess)      | ~5%           |
| **Gewinnmarge**                                      | **~3,9–4,2%** |

Die Branche ist **margenschwach** — der Gewinn liegt bei nur 3,9–4,2%. Das bedeutet: **Volumen und Auslastung** sind entscheidend. Jede unbesetzte Stunde kostet direkt Umsatz.

### Warum wechseln Pflegekräfte in die Zeitarbeit?

- **Höheres Gehalt** (oft 20–30% über Tariflohn der Einrichtung)
- **Planbare Arbeitszeiten** (keine spontanen Einspring-Dienste)
- **Mitsprache** bei Einsatzort, Schichten und Region
- **Wertschätzung** durch persönlichen Ansprechpartner (Disponent)
- **Abwechslung** durch verschiedene Einrichtungen

## 4. Interne Struktur eines Personaldienstleisters

### Typische Aufbauorganisation (mittelständisch, 50–200 ext. Mitarbeiter)

```
┌─────────────────────────────────┐
│        Geschäftsführung          │
│  (Strategie, Finanzen, AÜG-     │
│   Erlaubnis, Kundenbeziehungen)  │
└───────────┬─────────────────────┘
            │
   ┌────────┼──────────┬──────────────┐
   │        │          │              │
   ▼        ▼          ▼              ▼
┌──────┐ ┌────────┐ ┌──────────┐ ┌──────────┐
│Dispo-│ │Recruit-│ │Buchhaltung│ │ Vertrieb │
│sition│ │  ing   │ │& Personal-│ │(Kunden-  │
│      │ │        │ │verwaltung │ │ akquise) │
└──────┘ └────────┘ └──────────┘ └──────────┘
```

### Die Abteilungen im Detail

#### Disposition (Kernstück)

- Einsatzplanung und Schichtbesetzung
- Mitarbeiterbetreuung (1 Disponent betreut 40–80 externe Pflegekräfte)
- Ausfallmanagement (Krankmeldungen, spontane Absagen)
- AÜG-Compliance (Equal Pay Fristen, Überlassungshöchstdauer)
- Tägliche Kommunikation mit Pflegedienstleitungen der Kunden

#### Recruiting

- Stellenanzeigen (Indeed, Stepstone, Social Media)
- Bewerbungsgespräche und Qualifikationsprüfung
- Onboarding neuer Pflegekräfte (Unterlagen, Arbeitsvertrag, Einweisung)
- Active Sourcing (Direktansprache auf LinkedIn, XING, Facebook-Gruppen)

#### Buchhaltung / Personalverwaltung

- Lohn- und Gehaltsabrechnung (monatlich für alle ext. Kräfte)
- Rechnungsstellung an Kunden (Stundennachweise → Rechnung)
- Zeiterfassung (Stundennachweise prüfen und verbuchen)
- AÜG-Dokumentation (Verträge, Einsatznachweise, Erlaubnis)
- Mahnwesen bei säumigen Kunden

#### Vertrieb / Kundenakquise

- Neukundengewinnung (Kliniken, Heime, ambulante Dienste)
- Bestandskundenpflege und Rahmenverträge
- Preisverhandlungen (Verrechnungssätze)
- Marktanalyse (welche Einrichtungen suchen Personal?)

## 5. Der Disponent — Die Schlüsselrolle im Detail

Der Disponent ist die **zentrale Drehscheibe** des Personaldienstleisters. Er ist gleichzeitig Planer, Betreuer, Vertriebsmitarbeiter und Krisenmanager.

### Aufgabenbereiche

#### A) Einsatzplanung & Disposition

- **Schichtbesetzung:** Offene Anforderungen der Kunden mit verfügbaren Pflegekräften matchen
- **Dienstplanerstellung:** Gemeinsam mit den Pflegekräften Wochenpläne erstellen
- **Drag & Drop:** Im Dispo-Kalender (zvoove) Kräfte per Zeitstrahl auf Aufträge zuweisen
- **Qualifikations-Matching:** Examinierte Fachkraft vs. Pflegehelfer, Intensiv-Erfahrung, Sprachkenntnisse
- **Schichtmodelle:** Früh/Spät/Nacht, 8h/10h/12h Schichten, Wochenend-Dienste

#### B) Ausfallmanagement (die tägliche Feuerwehr)

- **Krankmeldungen** — Morgens ab 5:30 Uhr klingelt das Telefon: "Ich kann heute nicht"
- **Sofort-Ersatz finden** — Freie Kräfte abtelefonieren, WhatsApp-Gruppen, Springer-Pool
- **Kunden informieren** — Pflegedienstleitung anrufen: "Ersatz kommt um 7:00" oder "Kein Ersatz verfügbar"
- **Eskalation** — Wenn kein Ersatz: Schicht umverteilen, andere Kunden priorisieren

#### C) Mitarbeiterbetreuung

- **Persönlicher Ansprechpartner** für 40–80 Pflegekräfte
- **Onboarding:** Erste Tage am Einsatzort begleiten, Vorstellung beim Kunden
- **Probleme lösen:** Konflikte am Arbeitsplatz, Unzufriedenheit mit Schichten, private Themen
- **Motivation:** Wertschätzung zeigen, Feedback geben, Wünsche berücksichtigen
- **Arbeitszeiterfassung:** Stundennachweise prüfen, Überstunden genehmigen/ablehnen
- **Urlaubsplanung:** Urlaubswünsche koordinieren ohne Kundenversorgung zu gefährden

#### D) Kundenbetreuung

- **Bedarfsermittlung:** "Wie viele Kräfte brauchen Sie nächste Woche? Welche Qualifikation?"
- **Reklamationsmanagement:** "Die Kraft gestern war unpünktlich / passte nicht ins Team"
- **Rahmenvertrag-Pflege:** Verrechnungssätze, Konditionen, Verlängerungen
- **Beziehungspflege:** Regelmäßige Besuche vor Ort, Feedback-Gespräche mit PDL

#### E) AÜG-Compliance

- **Equal-Pay-Fristen** überwachen (9 Monate / 15 Monate pro Kraft pro Kunde)
- **Überlassungshöchstdauer** (18 Monate) tracken — rechtzeitig rotieren
- **Einsatznachweise** dokumentieren für die Bundesagentur für Arbeit
- **Arbeitsschutz:** Sicherstellen dass Kräfte PSA erhalten, Einweisungen bekommen

#### F) Recruiting-Anteil

- In kleineren Firmen übernimmt der Disponent auch Teile des Recruitings
- Bewerbungsgespräche führen
- Qualifikationen prüfen (Berufsabschlüsse, Anerkennungen, Führungszeugnisse)

### Typischer Arbeitstag eines Disponenten

| Zeit            | Tätigkeit                                                                   |
| --------------- | --------------------------------------------------------------------------- |
| **06:00–07:30** | Krankmeldungen bearbeiten, Ersatz organisieren, Kunden informieren          |
| **07:30–09:00** | Tagesplanung: offene Schichten prüfen, Einsätze für kommende Tage planen    |
| **09:00–11:00** | Telefonate: Kunden (Bedarfsabfrage), Pflegekräfte (Verfügbarkeit), Bewerber |
| **11:00–12:00** | Stundennachweise prüfen, Verwaltung, Equal-Pay-Fristen checken              |
| **12:00–13:00** | Mittagspause                                                                |
| **13:00–14:30** | Bewerbungsgespräche / Onboarding neuer Kräfte                               |
| **14:30–16:00** | Kundenbesuche vor Ort ODER Dienstplanung nächste Woche                      |
| **16:00–17:00** | Offene Themen abarbeiten, Nachtschicht-Besetzung finalisieren, Rückrufe     |
| **Ab 17:00**    | Rufbereitschaft (bei vielen Dienstleistern) für Nacht-/Wochenend-Ausfälle   |

### Die größten Herausforderungen

1. **Kurzfristige Ausfälle** — Krankmeldungen am Morgen, die sofort gelöst werden müssen. Via WhatsApp, Telefon und SMS nach Ersatz suchen — oft unter Zeitdruck
2. **Fachkraftquoten einhalten** — Nicht jeder darf jede Aufgabe übernehmen. Intensiv-Pflege braucht Fachkräfte, nicht Helfer
3. **Wünsche vs. Bedarf balancieren** — Pflegekraft will nur Frühdienst, Kunde braucht Nachtschicht
4. **Mehrere Kunden gleichzeitig** — Bei Engpass: Wen priorisiere ich? (Umsatz vs. Beziehung vs. Vertragspflicht)
5. **Emotionale Belastung** — Pflegekräfte haben oft persönliche Probleme, Kunden sind gestresst
6. **AÜG-Compliance** — Fristen vergessen = Bußgeld. Equal Pay nicht eingehalten = Nachzahlung
7. **Fluktuation** — Pflegekräfte wechseln häufig den Dienstleister wegen besserer Konditionen

### Skills eines guten Disponenten

- **Organisationstalent** — Dutzende parallele Vorgänge gleichzeitig managen
- **Kommunikationsstärke** — Verhandeln mit Kunden UND empathisch mit Pflegekräften
- **Stressresistenz** — Morgens um 6 Uhr Krisenmodus, den ganzen Tag Telefon
- **Pflegewissen** — Verständnis für Qualifikationen, Schichtmodelle, Stationsabläufe
- **Vertriebsaffinität** — Kunden gewinnen und halten
- **IT-Kompetenz** — Dispositionssoftware (zvoove), Excel, WhatsApp Business
